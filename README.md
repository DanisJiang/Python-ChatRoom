# QT-Python-ChatRoom
* 一个简单的Python-GUI聊天室

* 使用Python pyside2库

## Usage

1. 首先启动服务端 

```python
python3 server.py 127.0.0.1 port 
```

2. 启动客户端

```python
python3 client.py [client_host] [client_port] [server_host] [server_port] 
```
例如
```python
python client.py 127.0.0.1 2223 127.0.0.1 2222
```
